# Projeto de estudo

Projeto de estudos para simular transações

Ferramentas utilziadas: 

- typescript
- nodejs 18x
- fastify
- knex
- sqlite3
- zod
- cookie plugin
- eslint
- tsx
- vitest

# Start

- Clone o projeto;

- Acesse a pasta;

- Instale os pacotes ```npm install```

- Crie as migrations ```npm run knex -- migrate:latest```

- Rode o projeto com ```npm run dev```
